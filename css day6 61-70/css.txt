css是层叠样式表（Cascading Style Sheets）的简称，有时我们也会称之为CSS样式表或级联样式表。
css夜视一种标记语言。
|选择器| { 属性：值  属性：值}
选择器就是根据不同需求把不同标签选出来，简单来说，就是选择标签用的。

选择器分类：分为基础选择器和复合选择器。
   基础选择器又包括：标签选择器、类选择器、id选择器、通配符选择器。
（1）标签选择器：将某一类标签全部选择出来，可快速设置统一样式，但不能设计差异化样式。
<head>
    <style>
        p { 
            color: blue;
            }
        div {
            color: pink;
        }
    </style>
</head>
<body>
    <p>男生</p>
    <p>男生</p>
    <p>男生</p>
    <div>女生</div>
    <div>女生</div>
    <div>女生</div>
</body>

（2）类选择器(可多次使用)：可以差异化选择不同的标签，单独选择一个或者某几个标签。   结构需要class属性来调用class类的意思。
* 类选择器口诀： 样式点定义 结构类（class）调用  一个或多个 开发最常用 *
<head>
    …………
    …………
    <style>
	/* .后的命名是自定义的，不可用标签的类名如<p></p> <div></div>，不要使用纯数字、中文等命名，尽量使用英文字母来表示 */
        .red { 
            color: red;
        }
    </style>
</head>
<body>
    <ul>
        <li class="red">裴柱现</li>
        <li class="red">朴秀荣</li>
        <li>姜涩琪</li>
        <li class="red">孙胜完</li>
        <li>金艺琳</li>
    </ul>
</body>
	多类名使用方式：
		    <div class="yanse ziti beijing">这是一个盒子</div>
	多类名开发中的使用场景：
	① 可以把一些标签元素相同的样式（共同部分）放到一个类里；
	② 这些标签都可以调用这个公共的类，然后再调用自己独有的类；
	③ 从而节省CSS代码，统一修改也非常方便。

   （3）id选择器（一般用在页面唯一元素上）：可以为标有特定的id的HTML元素指定特定的样式。
	HTML元素以id属性来设置id选择器，CSS中id选择器以“#”来定义。
        /* id选择器的口诀： 样式#定义，结构id调用，只能调用一次，别人切勿使用 */

#id名{				例如：将id为nav的元素颜色设置为红色
	属性1：属性值1；			 #nav {
	……				     color: red;
   }					}

     （4）通配符选择器：在CSS中，通配符选择器使用“*”定义，他表示选取页面中所有元素（标签）。  *表示所有，把<html><head><body>这些标签都改了！不需要调用，自动就给所有标签都使用了样式。
<head >
    <style>
        *{
            font-size: 23px;
            color:green;
        }
    </style>
</head>
<body>
    <div>我的</div>
    <span>我的</span>  
    <ul>
        <li>还是我的</li>
    </ul>
</body>
